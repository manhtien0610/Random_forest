Mô tả dự án

Dự án ứng dụng Machine Learning sử dụng thuật toán Random Forest để dự đoán nguy cơ mắc ung thư phổi dựa trên dữ liệu sức khỏe bệnh nhân. 
Mô hình được xây dựng nhằm phân tích các yếu tố ảnh hưởng đến bệnh và hỗ trợ đưa ra dự đoán với độ chính xác cao.

Chức năng
- Đọc và xử lý dữ liệu bệnh nhân.
- Tiền xử lý và chuẩn hóa dữ liệu.
- Huấn luyện mô hình Random Forest.
- Dự đoán nguy cơ mắc ung thư phổi.
- Đánh giá hiệu suất mô hình bằng các chỉ số Accuracy, Precision, Recall và F1-Score.
- Trực quan hóa kết quả bằng biểu đồ.
Công nghệ sử dụng
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib.

Kết quả
- Xây dựng thành công mô hình dự đoán ung thư phổi bằng Random Forest.
- Đạt độ chính xác cao trên tập dữ liệu kiểm thử.
- Hỗ trợ phân tích các yếu tố nguy cơ liên quan đến bệnh ung thư phổi.
